This folder contains miniset_640px, which is a minimal dataset presented here for illustration of folder structures.
This shoud be placed in the folder:
MGR-Project-Code/Data/StreetViewData/
so that the path to Segments object is for example:

MGR-Project-Code/Data/StreetViewData/miniset_640px/SegmentsData_mark100.dump

See the hints of dataset usage in miniset_640px/dataset_usage_info.txt
